# shuge
仿书格网(https://new.shuge.org/) 设计的一个前后端分离PC端项目，采用 Vue+Express 开发，主要实现原网站首页UI、资源集分类展示古籍、添加展示评论等功能。

项目地址：https://github.com/unkkihere/shuge

### 项目技术栈
#### 后端技术栈
+ Express
+ Mongoose
+ MongoDB
+ ...(待补充)

### 前端技术栈
+ Vue2
+ axios
+ amcharts
+ vue-count-to
+ ...(待补充)

### 项目进度
#### 2022.09.18 
基本完成首页UI，后端图片接口的设计，其中图片存储在七牛云中
